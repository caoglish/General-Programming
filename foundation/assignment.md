# Assignment

## Primitive type assignment

### PHP

```php
$x="value";
```

### Python

```python
x="value"
```

### JavaScript

```javascript
var x="value";
```

## Array assignment

### PHP

```php
$x=["a","b","c"];
$x[3]="d";
```

### Python

```python
x=["a","b","c"]
x[3]="d";
```

### JavaScript

```javascript
var x=["a","b","c"];
x[3]="d";
```
