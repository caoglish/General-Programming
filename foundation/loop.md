# Loop

### Iterator(foreach):
#### Python

```python
for i in [1,2,3,4]:
	print(i)
```
#### PHP
```php
<?php
foreach([1,2,3,4] as $i){
	echo $i;
}
```
#### JavaScript
```javascript
for(let i in [1,2,3,4]){
  console.log(i);
}
```

### increment inner loop(for):
#### Python
```python
for i in range(0,6,1):
	print(i)
```
#### PHP
```php
<?php
for($i=0;$i<10;$i++){
	echo $i;
}
```
#### JavaScript
```javascript
for(let i=0;i<10;i++){
	console.log(i)
}
```
